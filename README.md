# portfolio
Personal Portfolio 
Overview
This is a modern, responsive, and interactive portfolio website built using HTML, CSS, Bootstrap, and JavaScript. It features structured sections for About, Experience, Education, Projects, Certifications, and Contact. The site showcases card-flipping animations, a Bootstrap carousel for featured work, and smooth scrolling for a professional user experience.​

Features
Responsive design: works seamlessly on mobile and desktop devices.

Navigation bar with smooth scrolling.

Interactive card-flipping animations for experience, education, projects, and certifications.

Bootstrap carousel for featured projects/work.

Tabs for combined credentials section.

Contact form with validation and instant feedback.

CSS animations for scroll and active navigation highlights.

Technologies Used
HTML5

CSS3 & custom CSS

Bootstrap 5
